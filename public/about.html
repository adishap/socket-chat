Let's Chat

Let's Chat provides chat room for users to send and receive free messages only till he is connencted to network. It will not keep history of your messages. You dont need to sign up for sending messages, just choose a username and get started.

This website is developed and maintained by Adisha Porwal. It is developed using Node.js, Socket.IO and Bootstrap. The site is deployed using Nginx on Ubuntu 14.04 Linux servers on VPS. This site uses a Git/GitHub for version control.
